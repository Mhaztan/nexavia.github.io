About statcalc

statcalc is a web application designed to help users visualize and analyze data. Built with HTML, CSS, Chartjs and JavaScript it provides a user-friendly interface for:

  • Uploading data files (CSV or manual data entry)
  • Generating various charts (bar charts, line charts, pie charts, doughnut charts, etc.)
  • Customizing chart appearance (size)
  • Exporting charts as images or downloadable files
  • settings (dark theme or light theme)
  • External Resources

Key Features

  • Responsive Design: statcalc adapts to different screen sizes for optimal viewing on desktops, tablets, and mobile devices.
  • Data Handling: The application supports various data formats and performs efficient data processing for visualizations.
  • Interactive Charts: Users can interact with the charts, such as zooming, and hovering over data points for detailed information.

Technical Stack

  • Frontend: HTML, CSS AND JavaScript/li>
  • Charting Library: Chart.js

  • Developed the data processing and visualization logic using traditional HTML, CSS and Chart.js.

  • Ensured the application is responsive and accessible across different devices.
  • Offline downloads and calculations/li>
